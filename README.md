# ai_script_rip
A lightweight Linux terminal tool for capturing AI chat output, cleaning clipboard text, and running prompt presets from an Obsidian vault. Designed for prompt experimentation, conversation logging, and building a portable personal AI knowledge base.
Another version is being designed with better UI, more well defined rules and ready to handle additional modules such as vault importer for bringing in your other vaults, and python scripts to connect your obsidian to an onboard AI like olamma.
