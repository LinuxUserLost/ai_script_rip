# ai_script_rip

## Breakdown
A lightweight Linux terminal tool for capturing AI chat output, cleaning clipboard text, and running prompt presets from an Obsidian vault. Designed for prompt experimentation, conversation logging, and building a portable personal AI knowledge base.  

## Original process
Designed to run on a cyberdeck build for my nephew, decided to expand for more powerful systems. 

###rules i started wth 
1 - Every script runs individually, one UI to bring it together for easy use and throw in plug and play.   
2 - minimal installs, currently just ktinker and python for debian and ubuntu.  
3 - Every script has individual terminal interface with easy menu navigation the functional navigation system incase your system just cant install ktinker or you need to debug. 
4 - Much more can be found in guides under /runtime/guides/ theres a few
